## 认证与授权

### 登录流程

- 短信验证码登录，验证码有效期5分钟
- JWT Token认证，有效期7天
- 敏感操作需二次验证


### 认证中间件

```python
from fastapi import Depends, HTTPException, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from jose import JWTError, jwt

security = HTTPBearer()

async def get_current_user(
    credentials: HTTPAuthorizationCredentials = Depends(security)
):
    token = credentials.credentials
    try:
        payload = jwt.decode(
            token,
            settings.JWT_SECRET,
            algorithms=[settings.JWT_ALGORITHM]
        )
        user_id = payload.get("sub")
        if user_id is None:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail="无效的认证凭证"
            )
        return user_id
    except JWTError:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="认证凭证已过期"
        )
```

### 用户数据保护

```python
# 敏感数据处理
def mask_phone(phone: str) -> str:
    """手机号脱敏"""
    return phone[:3] + "****" + phone[-4:]

def mask_sensitive_data(data: dict) -> dict:
    """脱敏敏感字段"""
    sensitive_fields = ["phone", "id_card", "bank_card"]
    result = data.copy()
    for field in sensitive_fields:
        if field in result:
            result[field] = mask_phone(result[field])
    return result
```

## 隐私保护

### 信息收集

- 手机号码：用于登录验证
- 音频样本：用于音色克隆
- 孕周信息：用于个性化内容生成

### 信息使用

- 手机号仅用于登录验证，不会用于营销
- 音频样本仅用于生成音色，不会用于其他用途
- 用户可以随时删除自己的音色和生成内容

### 数据删除

```python
# 用户数据删除接口
async def delete_user_data(user_id: str):
    """删除用户所有数据"""
    # 1. 删除音色
    await db.voice_samples.delete_many({"user_id": user_id})

    # 2. 删除生成记录
    await db.generated_contents.delete_many({"user_id": user_id})

    # 3. 删除订阅记录
    await db.subscriptions.delete_many({"user_id": user_id})

    # 4. 删除支付记录（保留必要信息用于财务）
    await db.payments.update_many(
        {"user_id": user_id},
        {"$set": {"user_deleted": True}}
    )

    # 5. 删除用户
    await db.users.delete_one({"_id": user_id})

    # 6. 清理OSS文件
    await oss_client.delete_folder(f"samples/{user_id}/")
    await oss_client.delete_folder(f"generated/{user_id}/")
```
