# AI亲音胎教微信小程序 - 实施计划索引

## 项目概述

AI亲音是一款面向孕妈的胎教微信小程序，核心功能包括：
- **声音克隆**：用户上传10秒音频样本，创建专属音色
- **内容生成**：AI生成胎教故事、日常对话等内容
- **语音合成**：使用克隆音色合成个性化胎教音频

## 技术架构

| 组件 | 技术选型 |
|------|----------|
| 前端 | 微信小程序原生开发 |
| 后端 | FastAPI + Motor (异步MongoDB) |
| TTS服务 | vllm-omni + Qwen3-TTS-12Hz-1.7B-CustomVoice |
| 部署 | 阿里云ECS + 本地TTS服务器(16GB VRAM) |

## 分阶段执行指南

### 执行方式

本计划支持按阶段独立执行，每个阶段完成后可进行评审确认再进入下一阶段。

```bash
# Phase 1 执行命令示例
请执行 Phase 1: 基础架构 (Tasks 1-5)

# Phase 2 执行命令示例
请执行 Phase 2: 核心业务 (Tasks 6-8)

# Phase 3 执行命令示例
请执行 Phase 3: 完善与部署 (Tasks 9-14)
```

### 阶段依赖关系

```
Phase 1 (基础架构)
    │
    ├── Task 1: 小程序骨架
    ├── Task 2: 后端骨架
    ├── Task 3: TTS服务 ──────┐
    ├── Task 4: 数据库设计    │
    └── Task 5: 微信登录      │
                             │
         ┌───────────────────┘
         ↓
Phase 2 (核心业务)
    │
    ├── Task 6: 用户/音色API ←── 依赖 Task 3, 4, 5
    ├── Task 7: 内容生成 ←────── 依赖 Task 6
    └── Task 8: 播放/历史 ←───── 依赖 Task 7
         │
         └───────────────────┐
                             ↓
Phase 3 (完善与部署)
    │
    ├── Task 9: 订阅支付 ←────── 依赖 Phase 2
    ├── Task 10: 个人中心
    ├── Task 11: 安全加固
    ├── Task 12: 测试调试
    ├── Task 13: 部署上线
    └── Task 14: 文档交付
```

### 里程碑检查点

| 里程碑 | 完成标志 | 验收标准 |
|--------|----------|----------|
| M1: 基础可用 | Phase 1完成 | 微信登录成功，TTS服务健康检查通过 |
| M2: 核心闭环 | Phase 2完成 | 用户可完成音色创建→内容生成→音频播放全流程 |
| M3: 生产就绪 | Phase 3完成 | 支付流程验证通过，安全检查清单全部勾选 |
